This walkthrough tries to simplify finding the link to course specific calendar feeds using the Canvas API. Since I can't pretend to be you when accessing Canvas, this walkthrough will give you links to click and will pick out the info you need based on the info you copy and paste from those links. None of this info is saved by me and all the logic is done right in your web browser.

Step 1: Make sure you're logged in to Canvas in this browser

Login to Canvas at canvas.uchicago.edu. Once you're logged in, come back to this page.

This method only works if you have cookies enabled. If you don't know what that means, you probably don't need to worry about it!
